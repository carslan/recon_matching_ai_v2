## Directory Layout
| Path | Purpose |
|------|---------|
| `workspace/knowledgebase/rulesets/{ws}/{rs}/` | KB files per ruleset |
| `workspace/outputs/{date}/{ws}/{rs}/{run_id}/` | Run output artifacts |
| `datasets/cust_of/` | Legacy ruleset.json for migration |
| `logs/{date}/` | Structured application logs |

## KB Files (per ruleset)
| File | Required | Format | Description |
|------|:---:|--------|-------------|
| schema_structure.json | Yes | JSON | Column inventory: base + computed, number_columns dict |
| features.json | Yes | JSON | Blocking strategies + amount_policy tolerance |
| core_matching_rules.json | Yes | JSON | Human-approved waterfall rules |
| agent_matching_rules.json | Yes | JSON | Agent-proposed rules (initially empty) |
| identity.md | Yes | MD | Agent mission + outputs |
| soul.md | Yes | MD | Behavioral guardrails + write paths |
| how_to_work.md | Yes | MD | Waterfall ordering + stop conditions |
| kb_index.json | Auto | JSON | Generated by kb_indexer (byte offsets, operator inventory) |

## Operator Types
| Operator | Comparisons Required | Aggregation | Blocking |
|----------|:---:|:---:|:---:|
| exclude_by_predicates | No | No | No |
| one_to_one_ranked | Optional | Not required | Optional |
| many_to_many_balance_k | Optional | Required | Optional |
| one_sided | Optional | Required | Optional |

## Amount Policy
Default tolerance: `0.005` (absolute). Set per-ruleset in `features.json` under `amount_policy.tolerance`.

## Stop Conditions
| Condition | Default | Description |
|-----------|---------|-------------|
| MAX_ZERO_PROGRESS_STEPS | 3 | Consecutive steps with 0 records removed |
| Pool empty | N/A | All records matched or excluded |
